GIT
1. Создание ветки  git checkout -B branch-name
-B - создание новой ветки.
branch-name - название ветки.

2. Добавление файлов к отслеживанию
git add .
. - все файлы в текущей директории.

3. Создание коммита.
 git commit -am "название коммита"
-am все файлы завернуть в коммит и присвоить название коммиту.

4. Отправка изменений в удлаенный репозиторий
git push  

5. Выгрузить изменения с удаленного репозитория.
git pull
